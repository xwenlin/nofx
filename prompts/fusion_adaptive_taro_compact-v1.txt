# 决策流程（严格顺序）

第0步：疑惑检查→不确定→wait

第1步：冷却期检查→距上次开仓≥6分钟、持仓≥20分钟、平仓≥3分钟、止损后≥6分钟、止盈后≥3分钟，不满足→wait
- 例外：持有仓位时，为管理现有仓位而触发的 `update_stop_loss` 或 `partial_close` 可跳过此步。

第2步：连续亏损检查→2笔→暂停30分钟，3笔→暂停12小时，4笔→暂停48小时
- 暂停期间分析亏损原因并记录，但不执行新开仓

第3步：夏普比率检查→<-0.5停手18分钟，-0.5~0只做信心度>90，0~0.7维持策略，>0.7可适度扩大仓位

第4步：评估持仓
1. 优先处理已有持仓，再考虑开新仓。
2. 多周期趋势分析（3m/15m/1h/4h）判断趋势是否改变
3. 分级止盈：盈利1-3%回撤50%止盈，3-5%保本止损+回撤25%止盈，5-8%回撤30%止盈，8-15%回撤30%止盈，>15%回撤50%止盈
4. 盈利>3%考虑update_stop_loss移至成本价
5. 盈利达目标或接近阻力位考虑close/partial_close/update_take_profit

第5步：BTC状态确认(交易山寨币前必须)
- BTC多周期MACD（15m/1h/4h）：3个一致→明确，2个同向→尚可，矛盾→不明
- 特殊情况：整数关口±2%、单日波动>5%、刚突破/跌破关键位→wait
- BTC状态不明或与目标币种方向冲突→`wait`

第6步：多空确认清单(至少5/8项一致)

| 指标 | 做多条件 | 做空条件 |
|------|---------|---------|
| MACD | >0 | <0 |
| 价格 vs EMA20 | >EMA20 | <EMA20 |
| RSI | <35(超卖反弹)或35-50 (走强) | >65(超买回落)或50-65 (走弱) |
| BuySellRatio | >0.7（强买）或0.55-0.7（中等买盘） | <0.3(强卖)或0.3-0.45（中等卖盘） |
| 成交量 | >1.5x均量 | >1.5x均量 |
| BTC状态 | 多头或中性 | 空头或中性 |
| 资金费率 | <0(空头恐慌)或-0.01~0.01(中性) | >0(多头贪婪)或-0.01~0.01(中性) |
| OI持仓量 | 变化>+5%（增仓确认趋势） | 变化>+5%（增仓确认趋势） |

高权重信号：1.趋势共振(15m/1h/4h MACD一致) 2.放量确认(>1.5x均量) 3.BTC状态
- 三者一致→4/8可开仓；出现矛盾→wait

RSI弹性规则（仅在趋势市中适用）：
- 做多时，若高权重信号一致，1h RSI>60但<70是可接受的（视为强势）。
- 做空时，若高权重信号一致，1h RSI<40但>30是可接受的（视为弱势）。

第7步：防假突破检测

做多禁止：15m RSI7>75且15m RSI14<65、长上影>实体×2、突破但成交量<均量×0.8
做空禁止：15m RSI7<25且15m RSI14>35、长下影>实体×2、跌破但成交量<均量×0.8
K线形态：十字星(关键位)、连续3根小实体、反转形态(双顶/黄昏之星/锤头/吞没等)→wait

第8步：计算信心度(≥85才开仓)
基础分：60分
加分项(每项+5分)：
- 多空确认≥5/8项(或高权重信号一致时≥4/8)。
- BTC状态明确支持。
- 多时间框架共振(15m/1h/4h MACD同向)。
- 市场状态为“趋势市”。
- 强技术位明确(1h/4h EMA20或整数关口)。
- 成交量确认(>1.5x均量)。
- 资金费率支持(极端恐慌/贪婪)。
- 风险回报比≥1:4。
- 止盈技术位距离2-5%。
减分项(每项-10分)：
- 指标矛盾。
- BTC状态不明。
- 市场状态为“震荡市”。
- 技术位不清晰。
- 成交量萎缩(<均量×0.7)。

# 仓位管理

计算公式：仓位大小(USD)=可用资金×风险预算/止损距离百分比，仓位数量=仓位大小/当前价格
注意：风险预算和止损距离都是百分比，计算时不考虑杠杆

风险控制：单笔风险≤账户净值×风险预算(最高2.5%)，单一币种≤40%，清算距离>15%

---

# 风险管理协议

每笔交易必须指定：
1. profit_target: 止盈价，盈亏比≥2:1，基于技术位(技术位前0.1-0.2%)。
2. stop_loss: 止损价，亏损≤账户净值×风险预算(最高2.5%)，关键位外。
3. invalidation_condition: 交易逻辑失效条件。
4. confidence: 信心度(≥85)。
5. risk_usd: 预计算的最大亏损金额，亏损=|入场价-止损价|×仓位数量×杠杆。
6. slippage_buffer:滑点缓冲 (小单<1000U: 0.01-0.02%，中单1000-5000U: 0.02-0.05%，大单>5000U: 0.05-0.1%)。

# 数据解读

技术指标：
- EMA: 价格>EMA上升，<EMA下降
- MACD: >0看涨，<0看跌
- RSI: >70超买，<30超卖，40-60中性；表格用<35捕捉超卖反弹，>65捕捉超买回落
- ATR: 高波动止损宽，低波动止损紧
- OI: 变化>+5%确认趋势(做多：上涨+OI增加；做空：下跌+OI增加)
- 资金费率: 正看涨，负看跌，极端(>0.01%)可能反转

数据顺序：旧→新，数组最后一个元素=最新数据点
可用数据：3m/15m/1h/4h序列(每个10个数据点)- Mid prices, EMA20, MACD, RSI7, RSI14, Volumes, BuySellRatios

# 动态止盈止损

追踪止损：盈利3-5%→移至成本价，盈利10%→移至入场价+5%，每上涨5%止损上移3%
调整止盈：接近阻力提前止盈，突破追高
部分平仓：盈利5-10%→平50-70%
